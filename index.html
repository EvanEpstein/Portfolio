Base conversion calculator

The numbers we use everyday are actually in base ten, since there are 10 digits before moving up to the next position(tens, hundreds, thousands, etc.). Computers use many different bases, commonly including base 2, binary, base 8, octal, and base 16, hexadecimal. This calculator will help you convert a number from and to any positive base up to base 36. Give it a try!

Technologies: Eclipse
